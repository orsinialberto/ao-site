# Cartella Immagini

Qui vanno le tue foto organizzate per categoria.

## Struttura delle cartelle:

```
images/
├── viaggi/
│   └── cover.jpg       (immagine principale per la homepage)
├── moto/
│   └── cover.jpg       (immagine principale per la homepage)
├── cammini/
│   └── cover.jpg       (immagine principale per la homepage)
└── portfolio/
    └── cover.jpg       (immagine principale per la homepage)
```

## Formato consigliato:

- **Formato**: JPG o PNG
- **Risoluzione**: 1200x900px (o superiore)
- **Peso**: < 1MB per foto (Next.js ottimizzerà automaticamente)

## Come aggiungere foto:

1. Crea una cartella per ogni categoria se non esiste già
2. Aggiungi le tue foto con nomi descrittivi
3. Assicurati che ogni categoria abbia almeno un `cover.jpg` per la homepage

Le foto verranno automaticamente ottimizzate da Next.js per performance migliori.
